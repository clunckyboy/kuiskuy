# KUISKUY
Kuiskuy adalah aplikasi mobile yang menyediakan kuis untuk dijawab oleh user. Kuiskuy memiliki fitur
memasukkan nama sebelum memulai kuis, mengerjakan kuis, serta menampilkan histori skor pengerjaan kuis
